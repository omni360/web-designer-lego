Utilitaire de construction Lego

Le but de cet outil est de réaliser une construction en Légo virtuelle.
Cette construction peut ensuite etre exporté vers un bras robot ( encore en développement).

Controles

  • Clic droit + mouvement : déplacement dans la scene
  • Clic millieu + mouvement : rotation de la scène
  • Roulette souris : zoom
  • C : mode création
  • D : mode suppression
  • R : rotation de la brique
  • Espace ou clic : action
  • H : afficher / masquer l'aide
